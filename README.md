#  YADRO Видеоплатформа

Современная веб-платформа для просмотра, загрузки и обмена видео с возможностью общения в реальном времени.

---

## Функциональность

- **Регистрация и авторизация** — безопасный вход в аккаунт
- **Загрузка видео** — делитесь своими видео
- **Просмотр видео** — удобная галерея с полноэкранным режимом
- **Чат в реальном времени** — общайтесь с другими пользователями
- **Личный кабинет** — управление своими видео
- **Удаление видео** — контролируйте свой контент
- **Адаптивный дизайн** — работает на всех устройствах

| Технология | Назначение |

## Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/anxssn/RGZ.git
cd RGZ
```
### 2. Создать виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate  # Для Mac/Linux
venv\Scripts\activate     # Для Windows
```
### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
python3 main.py
```
